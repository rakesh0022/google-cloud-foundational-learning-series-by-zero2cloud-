# Google Cloud Foundational Learning Series — Zero 2 Cloud


This repository contains companion materials for the YouTube series "Google Cloud Foundation — Zero 2 Cloud". Each `day-XX` folder contains notes, demo scripts, diagrams, and commands used in the video.


**How to use**
1. Pick a day/folder from this repo matching the video you are watching.
2. Follow the README for that day to reproduce demos locally or in your own GCP project.
3. Clone the repo and run the sample scripts under `demo-scripts/`.
